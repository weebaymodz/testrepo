Weebay Kodi Repository

Welcome to the Weebay Kodi repository.

Installation Instructions

  1. Open Kodi
  2. Go to Settings (gear icon) → System → Add-ons
  3. Enable "Unknown Sources"
  4. Go back to Settings → Add-ons
  5. Select "Install from zip file"
  6. Click "Add source"
  7. Enter: https://weebaymodz.github.io/repository.weebay
  8. Select the repository ZIP file
